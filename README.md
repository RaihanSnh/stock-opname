### Installation

1. Install package and dependencies
    ```sh
    npm install
    ```

2. Install tailwind 
    ```sh
    npm install tailwindcss postcss autoprefixer --save-dev
    ```

3. Copy the env.example file to .env 
   ```sh
   npm install react-router-dom
   ```

### Run Development Server

1. Run vite
    ```sh
    yarn dev
    ```

2. Run project
    ```sh
    npm run dev
    ```
